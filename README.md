# 🚀 Padrões de Projeto em Java

Este repositório contém a implementação de três importantes padrões de projeto utilizando Java.


## 📌 Padrões Implementados

### 🟢 Singleton

Garante que uma classe tenha apenas uma única instância e fornece um ponto global de acesso.


### 🟡 Abstract Factory

Permite criar famílias de objetos relacionados sem especificar suas classes concretas.


### 🔵 Bridge

Separa uma abstração de sua implementação, permitindo que ambas evoluam independentemente.
